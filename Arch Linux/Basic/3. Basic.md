# Hardware & Drivers

## AUR Helper (Yay)
```bash
git clone [https://aur.archlinux.org/yay.git](https://aur.archlinux.org/yay.git)
cd yay
makepkg -si
cd ..
rm -rf yay
```

---

# Software & Hyprland Ecosystem

## Multimedia Codecs
```bash
sudo pacman -S --noconfirm gstreamer gst-libav gst-plugins-base gst-plugins-good \
gst-plugins-bad gst-plugins-ugly ffmpeg
```

## Interface & Desktop Portal
```bash
sudo pacman -S --noconfirm hyprlock hypridle hyprcursor hyprpaper hyprpicker \
waybar rofi-wayland xdg-desktop-portal-hyprland xdg-desktop-portal-gtk \
dunst wl-clipboard polkit-kde-agent qt5-wayland qt6-wayland xdg-user-dirs-gtk

xdg-user-dirs-update
```

## Apps, Fonts & Utilities
```bash
# File Manager & Tools
sudo pacman -S --noconfirm dolphin dolphin-plugins ark kio-admin mpv pavucontrol

# Fonts
sudo pacman -S --noconfirm ttf-font-awesome ttf-opensans noto-fonts ttf-droid ttf-roboto

# AUR Utilities
yay -S --noconfirm hyprshot wlogout qview ttf-jetbrains-mono-nerd
```

# VSCode
flatpak install flathub com.visualstudio.code -y
