# Minimum Graphic Interface

## Update & Base Tools
```bash
sudo pacman -Syu --needed git flatpak fastfetch kitty hyprland
```

## Flathub & Browser
```bash
flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install flathub app.zen_browser.zen -y
```

```bash
reboot
```
