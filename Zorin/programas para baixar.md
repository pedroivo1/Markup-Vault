### Lista de Apps Baixados

* **Flameshot** (Ferramenta de captura de tela)
* **Grim** (Dependência instalada com o Flameshot)
* **Drawing** (Editor de imagens básico)
* **FFmpeg** (Motor de processamento de áudio e vídeo)
* **VLC Media Player** (Reprodutor multimídia universal)
* **Ghostscript** (Interpretador e compactador de arquivos PDF)
* **Nemo** (Gerenciador de arquivos alternativo com visualização de metadados)
* **LosslessCut** (Editor para cortes rápidos de vídeo sem renderização)
* **Python 3.13** (Versão atualizada da linguagem instalada via PPA Deadsnakes)
* **OneDrive Client** (Ferramenta de linha de comando para sincronização com a nuvem Microsoft)
* **Pip, Setuptools & Wheel** (Gerenciadores de pacotes essenciais do Python)
* **pympeg** (Biblioteca Python instalada no ambiente virtual para automação do FFmpeg)
* **software-properties-common** (Utilitário para gerenciar repositórios de software/PPAs)
* **wine**
